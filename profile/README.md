# 래플그라운드 Github

## 간단 소개
래플그라운드 깃허브입니다.
이곳에는 개발 소스코드가 전부 올라갈 예정입니다.

## 협업 방법
1. 리포지토리를 Clone한 후 브랜치를 생성합니다.
2. 브랜치에서 원하는 작업을 한 후 PR 요청을 하시면 함께 코드 리뷰를 하고 Main 브랜치로 Merge 합니다.
